Your complete streaming companion

StreamVault is a Fire TV and Android ITV (Internet Television) player that combines live TV, sports schedules, an EPG program guide, Plex integration, local/network media, and a recording system into one app.

Cache & Refresh

Repo playlists are cached for 7 days so the app opens fast without re-downloading megabytes of M3U data on every launch.

The ↻ refresh button on each repo card forces a fresh download right now. StreamVault also silently refreshes all repos once per day after 4am (in the background, no interruption).

🏆 Using Repos with Sports

In Settings → Live Sports, you can point a sport league at a repo group instead of your library. StreamVault will search that repo's streams when matching game cards — without those channels cluttering your Home screen.

Repos are not backed up in StreamVault's backup file (they can be 10 MB+). Only your repo names and URLs are saved. Re-adding the URL reconnects your repo immediately.

Auto-Failover

When a stream buffers beyond the timeout (default 15 seconds), StreamVault silently switches to the next failover URL. A small notification confirms the switch. Enable in Settings → Failover → Auto-failover.

Relay streams (from an HDHomeRun proxy) use an optimized 4-second buffer for smooth live TV playback. Non-relay streams use a longer 30-second buffer for timeshift stability.

🔁 Relay Proxy

If you run hdhr-xmltv (the companion relay server), StreamVault can route HDHR streams through it for deinterlacing, transcoding, and relay buffering. Enable in Settings → HDHR → Use Proxy Relay.

Relay URLs use a 4-second buffer profile optimized for live, encoder-paced output. This eliminates the buffering caused by irregular TS timestamps from interlaced broadcast sources.

🔌 USB Backup Injection (Fire TV)

You can silently push a StreamVault configuration to any Fire TV without opening the app UI. Useful for setting up multiple devices or recovering from a factory reset.

1
Export a backup from a configured StreamVault install: Settings → Backup & Restore → Export Backup. This saves a .json file.
2
On your computer, create a folder called inject in the root of a USB drive. Place the .json backup file inside it:
USB Drive/
  inject/
    mybackup.json
3
Plug the USB drive into the Fire TV's USB port (use a USB-A hub if needed — Fire TV Stick requires a powered hub).
4
In StreamVault, go to Settings → Backup & Restore → Inject from USB and tap the button. StreamVault scans all connected storage volumes for the inject folder.
5
If a backup is found, StreamVault confirms the filename and asks whether to apply it. Tap Apply to restore the full configuration immediately.
StreamVault tries three detection strategies on Fire TV: StorageManager API, external app directories, and direct scan of /mnt/media_rw/ — the Fire TV's USB mount point. If inject is not found, the error message lists every path checked.
Fire TV Stick 4K and Lite models require a powered USB hub. Fire TV Cube has a full USB-A port and works without a hub.
Tips & Tricks
🏈 Sport Stream Matching

StreamVault matches game cards to streams by looking for both team names (or abbreviations) in the stream's M3U name. If your provider updates stream names daily to reflect today's games, the daily 4am background repo refresh ensures those names are always current by the time you open the app in the morning.

⚡ Startup Speed

Repo stream data is cached for 7 days. EPG data is parsed in the background after launch. The splash screen hides once repos load — EPG continues silently so you can start browsing immediately.

📡 Relay Streams (hdhr-xmltv)

Relay URLs (containing /proxy/) are automatically detected and given a special 4-second buffer profile. This is optimized for encoder-paced live streams like your HDHomeRun broadcast channels and prevents the micro-stutter that a standard 30-second buffer would cause.

📺 Fire TV Performance
🔒
D-pad navigation is locked during background operations (EPG fetch, repo refresh) to prevent ghost keypresses from crashing navigation.
💾
EPG parsing uses a chunked, async algorithm that handles 32,000+ channel EPG feeds without running out of memory.
🗺
Sport game matching uses pre-built index Maps (not nested loops) so even large repos match instantly.
🔧 Troubleshooting
?
Stream won't play — try the ↻ refresh button on the stream, or check the URL is still valid. Check if failover is configured.
?
No EPG data — verify your XMLTV URL is reachable. Check that stream EPG IDs match channel IDs in the file.
?
Sport games not matching — refresh the repo manually (the stream name may have been updated by your provider today). Check the league channel group setting in Settings → Live Sports.
?
Storage full warning — use URL-based repos instead of file imports. Large M3U data stays server-side.
Plex / PlexStr

PlexStr lets you browse and play your Plex Media Server libraries directly inside StreamVault. You need your server URL and a Plex authentication token.

🔑 Step 1 — Find Your Plex Token
Web
Plex Web (easiest): Open app.plex.tv. Sign in. Click any title. In the address bar find X-Plex-Token=YOUR_TOKEN and copy everything after the equals sign.
XML
XML method: While signed in, navigate to https://plex.tv/api/v2/user/access-token — the response contains your token in the authToken field.
Dev
DevTools: Press F12 → Network tab → Filter XHR → click any request → look for X-Plex-Token in the request headers.
🌐 Step 2 — Find Your Server URL
LAN
Local: http://192.168.x.x:32400 — your Plex server's local IP, port 32400.
WAN
Remote: Plex server settings → Remote Access gives you a plex.direct URL.
Test
Append /web to the URL in a browser — if Plex Web loads, the URL is correct.
⚙ Step 3 — Connect in StreamVault
1
Go to Settings → Plex / PlexStr.
2
Enter server URL (include port: http://192.168.1.50:32400) and paste your token.
3
Tap Test Connection — libraries appear in a dropdown on success.
4
Select a library and tap Save. A PlexStr tab appears on the Home screen.
5
Tap Add Plex Stream to browse and add titles to your library.
Keep your Plex Token private — it grants full access to your server. Rotate it any time in Plex Account Settings → Authorized Devices.

VOD & PlexStr

VOD (Video on Demand) lets StreamVault discover movies and TV shows inside your ITV repos and display them in a Netflix-style grid with posters, ratings, and cast. PlexStr puts your personal Plex library in the same place.

🎬 Movies & TV Shows Tabs

The VOD page has three tabs — Movies, TV Shows, and PlexStr (only shown when Plex is configured). Each tab has scrollable content rows. Use D-pad left/right to move between tabs, D-pad up/down to move between rows and cards.

🎬
Movies tab: repo streams classified as movies, sorted into genre rows (Action, Comedy, Drama, Horror, etc.). Continue Watching row appears when you have watch progress.
📺
TV Shows tab: repo streams classified as series, grouped by show and season. Navigate episodes with D-pad.
🎬
PlexStr tab: all Plex movies and shows you've added, shown in a full-page poster grid. Tap/select to play directly through the native Plex player.
🔍 How VOD Content is Detected
1
StreamVault scans your repos for streams with tvg-type="movie", tvg-type="series", or titles that match the movie/series keyword lists.
2
Detected streams are grouped by title (duplicates from different quality variants are merged). The highest-quality URL is kept.
3
Genre is inferred from the group name and title using built-in keyword maps and a title lookup table for popular films.
4
If a TMDB API key is configured in VOD Settings → General → TMDB, posters, ratings, synopsis, cast, and runtime are fetched and cached locally (no repeated lookups).
⚙ VOD Settings (gear icon top-right of VOD page)

Tap the gear icon while on the VOD page to open VOD-specific settings. The settings page has three tabs: Movies, TV Shows, PlexStr.

🔑
General → TMDB API Key: free key from themoviedb.org. One key covers Movies, TV Shows, and Plex. Posters load immediately for matching titles and are cached in sv_tmdb_cache.
☰
Visible Rows → Edit Rows & Order: opens a compact modal. Toggle rows on/off with the toggle on the right. Use ▲ ▼ to reorder. Press Back or Apply when done. Order is saved per type (movies/shows).
📁
VOD Group Management → Scan: scans all repos and auto-detects which channel groups contain movies/series. Live channels are always excluded. You can also manually add groups not auto-detected.
📦
Standalone VOD Repo: a dedicated M3U URL just for VOD content. Faster than scanning all ITV repos. Enable "Use Only This Repo" to skip ITV repos entirely for VOD.
🎬 PlexStr in VOD
1

💾 Caching Summary
🎬
TMDB data (posters, ratings, synopsis, cast) is cached per title in sv_tmdb_cache. Cache is in memory during a session and saved to storage with a 1.5-second debounce. Re-fetched automatically only when a title has no synopsis yet.
📡
M3U stream lists are cached per repo in sv_m3u_cache_*. Re-fetched when "Refresh Content" is tapped or when the repo cache expires.
🎬
Plex posters come from your Plex server directly (authenticated URLs). They are not cached locally — your Plex server handles caching.
📋 Watch History & Continue Watching

StreamVault tracks your viewing progress for VOD content. The Continue Watching row appears automatically at the top of the Movies/Shows tab when you have unfinished titles. Remove a title from Continue Watching with the ✕ button on its poster card (appears when the card has focus on TV, on hover on mobile). Full history is in VOD Settings → Watch History, split by Movies, TV Shows, and PlexStr.
